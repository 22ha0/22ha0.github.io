Abstract

摘要部分提出了更深层次的神经网络更难被训练。并提出了一种残差学习框架，以简化网络的训练，使得网络的层数能够更深。在ImageNet数据集上，本文使用的ResNet深度高达152层是VGG网络的8倍，但仍然具有较低的复杂度。这些残差网的集合在ImageNet测试集上达到了3.57%的错误率。该结果在ILSVRC 2015年分类任务中获得了第一名。（简要介绍网络，主要是介绍取得的成功）

Introduction

介绍中强调了网络的深度是很重要的。提出了学习更好的网络就像堆叠更多的层次一样容易吗？在层数加深时可能会出现梯度消失或者梯度爆炸的问题。虽然这个问题已经被归一化初始化和中间归一化层所解决，但是当更深层次的网络能够开始收敛时，一个退化问题就暴露出来了：随着网络深度的增加，精度达到饱和（这可能并不奇怪），然后迅速下降。出乎意料的是，这种退化不是由过拟合引起的，在适当的深度模型中添加更多的层会导致更高的训练误差。

使用20层和56层“普通”网络的CIFAR-10上的训练误差（左）和测试误差（右）。较深的网络具有较高的训练误差，因此存在测试误差。

残差结构（本文提出的最重要的结构）

本文通过引入一个深度残差学习框架来解决退化问题。不是希望每几个堆叠层直接匹配所需的底层映射，而是明确地让这些层匹配残差映射。

将期望的底层映射表示为H (x)，我们让堆叠的非线性层适合F (x)=H(x)−x的另一个映射。原始映射被转化为F (x)+x。

这里也就是说，原始的思想是对H(x)进行学习，但是ResNet提出的残差网络结构是对F(x)进行学习，也就是原始的H(x) - x。最后往下一层输出的时候再让F(X) + x。

F (x) + x的公式可以通过具有“shortcut connections”的前馈神经网络来实现（图2）。

作者在ImageNet上进行实验验证

在ImageNet上作者进行了全面的实验来表明退化问题，并评估了我们的方法。

结果表明：

1)本文提出的极深残差网很容易优化，但是对应的“plain”网络表现出比本文网络更高的训练误差；

2)本文的深度残差网可以很容易地从大大增加的深度中获得精度，其结果明显优于以前的网络。

Deep Residual Learning

Residual Learning

多层神经网络（不一定是整个网络）可以看做是一个隐含映射，我们把它记作H(x)，x表示其中第一层的输入。如果假设多个非线性层可以渐近近似复杂函数，那么它就等价于假设它们可以渐近近似残差函数，即H (x)−x（假设输入和输出具有相同的维数）。因此，我们不是期望堆叠的层近似于H (x)，而是明确地让这些层近似于一个残差函数F (x) = H (x)−x。原来的函数就变成了F (x)+x。虽然这两种形式都应该能够渐近地近似于所期望的函数（如假设的那样），但学习的容易程度可能是不同的。

Identity Mapping by Shortcuts

这里的x和y是所考虑的层的输入和输出向量。函数F（x，{Wi}）表示要学习的残差映射。对于图2中有两层的例子，F = W2σ（W1x），其中σ表示ReLU，为了简化符号，省略了偏差。操作F + x是通过一个快捷方式连接和元素级加法来执行的。我们采用了加法后的第二个非线性关系（即σ(y)，见图2）。

这种操作既没有引入参数也没有带来额外的计算复杂度。这不仅在实践中具有吸引力，而且在我们比较普通网络和残差网络时也很重要。我们可以公平地比较同时具有相同数量的参数、深度、宽度和计算成本（除了可以忽略的逐元素加法）的普通/残差网络。

x和F的尺寸必须相等。如果不是这样（例如，当改变输入/输出通道时），我们可以通过快捷连接执行一个线性投影Ws来匹配维度： y = F（x，{Wi}）+ Wsx。

我们也可以在方程(1)中使用方阵。但我们将通过实验表明，恒等映射足以解决退化问题，并且是经济的，因此Ws只在匹配维数时使用。

残差函数F的形式是灵活的。本文的实验涉及一个函数为两层或三层的函数F（图5），同时可能有更多的层。但如果F只有一个层，Eqn。(1)类似于线性层： y = W1x + x，我们没有观察到其优势。

我们还注意到，尽管为了简单起见，上述符号是关于全连接层的，但它们也适用于卷积层。函数F（x，{Wi}）可以表示多个卷积层。元素级添加将在两个特征映射上执行，一个通道一个通道。

Implementation

随机缩放，然后其中及其水平翻转中随机采样224*224的尺寸，减去每个像素的均值，然后使用标准颜色增强。在卷积之后、激活之前使用batch normalization (BN) 。权重初始化好像用了Kaiming initialization。用了SGD，LR从0.1开始，每当误差平稳时便除以10，整个迭代次数高达60x10 4次方。weight decay=0.0001，momentum=0.0001，没有使用Dropout。
