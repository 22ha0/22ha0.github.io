跨平台：可以在任意操作系统上运行

JRE和JDK

JRE （JavaRuntime Environment）Java程序运行时的程序

JDK （Java Development Kit) Java 开发工具包 包含JRE和开发人员使用的工具

编译工具：javac.exe 运行工具：java.exe

只需要JDK就可以运行Java程序
